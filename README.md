## Про модуль

### Цей модуль допомагає закріпити практичні навички роботи з DOM та подіями в JavaScript:

• Допомагає розуміти, що таке об’єктна модель документа (DOM) і як будується ієрархія DOM-елементів.<br>
• Допомагає визначати та використовувати властивості та атрибути DOM-елементів.<br>
• Допомагає маніпулювати DOM деревом, створюючи та видаляючи елементи.<br>
• Допомагає розуміти сутність подій у браузері та принцип їх роботи.<br>
• Допомагає обробляти події за допомогою методів addEventListener() та removeEventListener().<br>
• Допомагає застосовувати події елементів форм для взаємодії з користувачем.
